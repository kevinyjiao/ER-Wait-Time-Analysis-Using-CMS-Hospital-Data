# CMS Emergency Department Wait Time Analysis

This project analyzes hospital emergency department wait time measures from the Centers for Medicare & Medicaid Services (CMS). It combines a hospital-level wait time file with a hospital general information file to answer:

- Which states have the longest average ER wait times?
- How do hospital characteristics such as size, type, and ownership relate to ER wait times?
- How do urban and rural hospitals compare?

The output is a set of clean charts and summary tables that you can use to write your own one-page PDF or presentation slide.

## Project Structure

```text
data/
  raw/              Put the two CMS CSV files here
  processed/        Cleaned analysis outputs
outputs/            Charts generated by the analysis
src/
  er_waittimes_analysis/
    analysis.py     Main data cleaning, analysis, and charting code
run_analysis.py     Command-line entry point
requirements.txt    Python dependencies
```

## Data Setup

Put your two CSV files in `data/raw/`.

Expected files:

1. A CMS hospital wait time / timely and effective care file
2. A CMS hospital general information file

You can name them anything. The script will inspect the columns and try to identify which file is which. If it cannot, pass the paths explicitly:

```bash
python run_analysis.py --wait-file "data/raw/your_wait_times.csv" --info-file "data/raw/your_hospital_info.csv"
```

## How To Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the full analysis:

```bash
python run_analysis.py
```

## Current CMS Run

The included workflow was tested against the CMS Provider Data Catalog hospital files released on May 13, 2026:

- `Timely and Effective Care - Hospital`
- `Hospital General Information`

The analysis selected the CMS emergency department measure:

> Average median time patients spent in the emergency department before leaving from the visit, excluding patients transferred to another facility or psychiatric care/mental health patients.

Key results from this snapshot:

- The analysis included 4,077 hospitals with usable ER wait-time values.
- The longest average state-level waits were in DC, PR, MD, MA, and RI.
- Acute care hospitals averaged about 175 minutes, compared with about 116 minutes for Critical Access Hospitals.
- Urban/other proxy hospitals averaged about 175 minutes, compared with about 116 minutes for the rural Critical Access proxy group.
- Medium-volume hospitals had the highest average wait time among the volume-based size groups.

## Notes On Methodology

The analysis focuses on emergency department measures. The script prioritizes median time spent in the ED before leaving, usually CMS measure `OP_18b`, when that measure is available. If not, it uses the first emergency department wait time measure it can identify from the CMS measure names or IDs.

Hospital size is taken from an explicit bed-count column when available. If the general information file does not include beds, the script uses the CMS wait time sample count as a volume-based proxy and labels hospitals as small, medium, or large.

Urban/rural status is taken from an explicit urban/rural column when available. If unavailable, the script uses a conservative proxy: Critical Access Hospitals are classified as rural, and other hospitals are classified as urban/other. Label this limitation clearly if you use the urban/rural chart in a slide.

## Interview Talking Points

- The analysis turns raw CMS hospital data into an operations-focused view of ER throughput.
- The strongest state-level findings show where patient flow problems may be most visible.
- Ownership, hospital type, and size comparisons help identify whether delays are concentrated in certain operating models.
- A next step would be to add staffing, bed occupancy, local demand, ambulance diversion, and county-level urban/rural classifications.
